The PostgresCompare CLI is a planned feature and is not yet available.

## Coming Soon

When released, the CLI will support commands for:

- **Schema comparison** between PostgreSQL databases
- **Deployment script generation** from comparison results
- **Snapshot management** for offline and historical comparisons

## Current Option

All comparison and deployment features are available in the PostgresCompare desktop application. See the [Quick Start Guide](/docs/getting-started/quick-start/) to get started.

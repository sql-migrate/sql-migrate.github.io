The PostgresCompare CLI is a planned feature and is not yet available. Examples for CLI usage, including CI/CD integration, will be added when the CLI is released.

## Coming Soon

Planned examples will cover:

- Basic schema comparison from the command line
- CI/CD pipeline integration (GitHub Actions, GitLab CI, Jenkins)
- Automated snapshot workflows
- Multi-environment comparison scripts

## Current Option

All comparison and deployment features are available in the PostgresCompare desktop application. See the [Comparing Databases Guide](/docs/guides/comparing-databases/) for detailed usage.
